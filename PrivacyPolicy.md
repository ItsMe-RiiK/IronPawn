This Privacy Policy explains how IronPawn collects, uses, and protects user data when used on the Discord platform.

1. What Data Do We Collect?
To run the chess game and its features, IronPawn collects only the following minimum data points:
    - Discord User ID: Used to identify players, track the status of ongoing games.
    - Discord Guild (Server) ID: Used to store bot settings specific to that server.
    - Messages: If the bot uses prefix commands (e.g., !play), it will read the message to process the command. The bot does not record or store regular chat messages.

2. How Is the Data Used?
The data collected is used exclusively for the bot’s core functionality, namely:
    - Starting and processing chess games between users.
    - Storing and displaying leaderboards or player statistics.

3. Data Sharing
We do not sell, rent, or share user data with any third parties. Data is processed solely by the bot for gameplay purposes.
